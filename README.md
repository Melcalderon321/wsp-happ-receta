# Happ - WhatsApp Flow Demo 🚀

Este proyecto es un demo visual interactivo diseñado para mostrar el flujo de cotización de recetas médicas a través de **WhatsApp Flows** para la plataforma **Happ**.

## ✨ Características
- **Interfaz Realista**: Diseño fiel a WhatsApp (iOS style) optimizado para dispositivos móviles.
- **Flujo Interactivo**: Simulación completa desde el saludo inicial hasta la aplicación de beneficios.
- **WhatsApp Flows**: Implementación de tarjetas nativas de orden y modales de detalle de productos.
- **Rápido y Dinámico**: Animaciones y tiempos de respuesta ajustados para una experiencia fluida.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Despliegue listo para Vercel

## 🚀 Instalación y Despliegue

### Local
Simplemente abre el archivo `index.html` en cualquier navegador moderno.

### Despliegue en Vercel
1. Sube esta carpeta a un nuevo repositorio en GitHub.
2. Conecta tu cuenta de Vercel con GitHub.
3. Importa el repositorio y ¡listo! El proyecto se desplegará automáticamente.
